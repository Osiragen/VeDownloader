VeDownloader Portable
===================

This is a portable version of VeDownloader that can run on Ubuntu without installation.

How to use:
-----------
1. Extract this directory to any location on your Ubuntu system
2. Make sure the launcher script is executable:
   chmod +x VeDownloader-Portable.sh
3. Run the launcher script:
   ./VeDownloader-Portable.sh

Requirements:
------------
- Ubuntu or compatible Linux distribution
- Python 3.6 or higher (included in the package)
- FFmpeg (included if available during package creation)

Notes:
------
- All dependencies are included in the package
- No installation is required
- You can move this directory anywhere on your system
- You can run the application from a USB drive

If you encounter any issues, please check the README.md file for troubleshooting.
